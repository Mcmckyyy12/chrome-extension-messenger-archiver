# 💬 Messenger Auto Archiver

A Chrome Extension that automatically archives all your Facebook Messenger chats in one click. No logins stored, no data collected — runs entirely in your browser.

---

## Requirements

- Google Chrome (v88 or later)
- Logged into [messenger.com](https://www.messenger.com)

---

## Installation

1. Download and extract `messenger-archiver-v2.zip`
2. Open Chrome and go to `chrome://extensions`
3. Toggle **Developer mode** ON (top-right corner)
4. Click **Load unpacked** and select the `messenger-archiver` folder
5. Pin the extension to your toolbar for easy access

---

## How to Use

1. Go to [messenger.com](https://www.messenger.com) and log in
2. Click the **Messenger Auto Archiver** icon in your toolbar
3. Set the **Max chats** limit (default: 9999 = archive everything)
4. Click **▶ Start** and watch it run
5. Click **⏹ Stop** anytime to pause

---

## Troubleshooting

**"Open messenger.com first!"**
Make sure you are on messenger.com and fully logged in before clicking Start.

**Many chats skipped**
Your page may be loading slowly. Wait for the chat list to fully load before starting, or try again on a faster connection.

**Extension not appearing**
Make sure you selected the `messenger-archiver` folder (the one containing `manifest.json`), not a file inside it.

**Stats not updating**
Keep the popup open while it runs. Closing the popup stops the live display, but archiving continues in the background.
